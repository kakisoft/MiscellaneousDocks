## エラー発生時
```
Format '"%s" の不明なデータ型 oid #%d です。%s にフォ' invalid or incompatible with argument.
```
回避策
```
[設定]タブ→続き→環境設定→言語→『English』に変更。

※再起動後に有効
```
参考サイト：<https://github.com/HeidiSQL/HeidiSQL/issues/55>
