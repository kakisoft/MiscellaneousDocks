## キーボードの記号

|  記号  |  名称     |
|:-------|-----------|
|  ⌘    |  command       |
|  ⌥    |  option(Alt)   |


## 日本語 ⇔ 英語 変換
US キーボード
```
control + space
```

## ターミナル起動
command + space ⇒ 「ter」


## ファイル削除
command + delete


## ファイルを開く
command + ↓


## コピペ
command + c ⇒ command + v


## 最大化
command + control + F


## 中断（ターミナル）
control + C


## デスクトップを表示
fn + F11


## 画面キャプチャ（全体）
command + shift + 3

## 画面キャプチャ（範囲指定）
command + shift + 4

## ペイント
プレビュー（というツール）  
上段メニューの「A」みたいなマークを押すと、編集系のメニューが出てくる。
