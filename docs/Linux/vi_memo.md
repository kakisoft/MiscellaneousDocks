```
【 vi 】
＜undo（１つ前の状態に戻る）＞
u

＜redo（undoの逆）＞
ctrl + r

＜検索＞
「/」入力後、検索文字列を入力。
N を入力で次にヒットした位置に移動。
検索文字には正規表現使用可。

＜コピーペースト＞
コピー：「v」→矢印キーで範囲指定
ペースト：[Shift] + p

ペーストは、ノーマルモードに戻って p でもＯＫ

＜1行削除＞
dd

＜1行コピー＞
yy

＜ペースト＞
p

＜移動＞
^ 行頭
$ 末尾

f + 何かの文字

＜{ 移動＞
{ の上で % で、対応する } に移動


＜ビジュアルモード＞
v 文字単位のビジュアルモード
V　行単位
ctrl V 矩形単位

＜全選択＞
gg V → shift G

```
