```
【 vi 】
＜undo（１つ前の状態に戻る）＞
u

＜redo（undoの逆）＞
ctrl + r

＜検索＞
「/」入力後、検索文字列を入力。
N を入力で次にヒットした位置に移動。
検索文字には正規表現使用可。

＜コピーペースト＞
コピー：「v」→矢印キーで範囲指定
ペースト：[Shift] + p

ペーストは、ノーマルモードに戻って p でもＯＫ

＜1行削除＞
dd

＜1行コピー＞
yy

＜ペースト＞
p

＜先頭へ移動＞
gg 

＜末尾に移動＞
G 

＜画面単位の移動＞
Ctrl f 　（下方向）
Ctrl b   （上方向）

＜単語単位での移動＞
w   単語単位で進む
b   単語単位で戻る

＜行頭へ移動＞
^

＜行末に移動＞
$

＜特定の文字へ移動（行単位）＞
f + 何かの文字

＜{}移動＞
{ の上で % で、対応する } に移動
```
