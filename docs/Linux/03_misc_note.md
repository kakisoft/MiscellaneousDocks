## .bash_profile
.bash_profile はログイン時にのみ実行されます。具体的な用途は:  
　  
環境変数を設定する (export する変数)  
環境変数はプロセス間で勝手に受け継がれるのでログイン時のみ設定すれば十分です。  


## .bashrc
.bashrc は対話モードの bash を起動する時に毎回実行されます。具体的な用途は:

 * 環境変数でない変数を設定する (export しない変数)
 * エイリアスを定義する
 * シェル関数を定義する
 * コマンドライン補完の設定をする
 
 これらは bash を起動する度に毎回設定する必要があるものです。
