
| ディレクトリ |    内容                                                                            |
|:-------------|:-----------------------------------------------------------------------------------|
|  /bin        |  cat や cp といったシステムを動かすための基本コマンド                              |
|  /boot       |  カーネルイメージ                                                    |
|  /dev        |  デバイスファイル                                                    |
|  /etc        |  アプリケーションの設定ファイル                                                    |
|  /home       |  各一般ユーザのホームディレクトリ                                                      |
|  /sbin       |  システム管理用のコマンド                                                          |
|  /user       |  一般的なアプリケーション。新しくアプリをインストール場合、大抵ここに入れられる。  |
|  /var        |  ログ、メール、プリントスプール etc...    頻繁に書き換わるファイルが置かれる。                  |
