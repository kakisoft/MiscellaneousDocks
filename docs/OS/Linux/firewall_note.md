## CentOS 7
```
#### サービスの状態確認
systemctl status firewalld
systemctl status firewalld.service

　・Active: active (running)
　・Active: inactive (dead)


#### 起動
systemctl start firewalld.service


#### 終了
sudo systemctl stop firewalld.service

```
