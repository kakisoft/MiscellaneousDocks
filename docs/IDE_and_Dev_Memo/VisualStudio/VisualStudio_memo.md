#### ショートカット

|　　選択                               |　　キー                    |
|:--------------------------------------|:---------------------------|
|  大文字に変換                         |  Ctrl + Shift + u          |
|  小文字に変換                         |  Ctrl + u                  |
|  フォルダを指定して検索               |  Ctrl + Shift + f          |
|  クイックウォッチ                     |  Ctrl d → q               |
|  リネーム                             |  選択→ Ctrl + r → r      |
|  メソッド切り出し                     |  選択→ Ctrl +  r → m     |
|  ファイル名を検索して開く               |  Ctrl + ,（カンマ）     |
