## 設定
```
Cyberduck
編集→環境設定→[ブラウザ]タブ→
　・'.' で始まるファイルを表示
　・ ダブルクリックしたファイルを外部エディタで開く

外部エディタ
　→適当なエディタを設定
```

## ssh over FTP
```
「SFTP」を選択

ポートは ssh と同じ。

鍵を選択
```


## S3への接続
サーバ ： s3.amazonaws.com  
ポート : 443  
Path ： バケット名  
アクセスキーID ： <YOUR_ACCESS_KEY>  
シークレットアクセスキー ： <YOUR_SECRET_ACCESS_KEY>  
　  
左下にも、接続情報追加のボタンがある。
