# Google Chrome

## ソースを表示
command + option + u

## デベロッパーツール起動
command + option + i

## Webページ内の要素を検証
デベロッパーツールの左上のマークを押す -> ページの要素を選択

# Elements タブ

### 編集
```
移動、削除、編集、id や class を変更
右クリック -> Edit as HTML
```

### css/styke
```
Elements  -> Styles 
タブにて、css の内容を確認できる


css ソースファイル名も確認可。
```
