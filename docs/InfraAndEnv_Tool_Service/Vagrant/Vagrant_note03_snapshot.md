## スナップショットの一覧
```
vagrant snapshot list
```

## スナップショット作成
「savepoint1」という名称で作成。
```
vagrant snapshot save savepoint1
```

## スナップショット復元
```
vagrant snapshot restore savepoint1
```

## スナップショット削除
```
vagrant snapshot delete savepoint1
```
