NuGetで「SQLite」

## System.Data.SQLite
全部入り。    
Entity Frameworkや、SQLiteのLINQ拡張などが全部入ったセット。    
実行ファイルのディレクトリにファイルがいっぱい増える。    
```
ツール→NuGetパッケージマネージャー→パッケージマネージャーコンソール

Install-Package System.Data.SQLite
```
